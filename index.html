This project is a Python-based web scraper designed for the "Books to Scrape" website. It efficiently extracts key book details such as titles, prices, ratings, and availability, storing the data in a CSV file. The scraper is user-friendly, allowing customization of the number of pages to scrape and the output file name. It features robust error handling and logging for reliability and ease of use.
